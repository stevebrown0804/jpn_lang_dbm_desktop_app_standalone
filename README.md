Reminder: This project currently depends on there being an instance of jpn-lang-db-manager listening on port 50000.

Also, the program will create the DB on first load (and any other load, if you delete it), so don't worry about it existing beforehand.

You may need the following fonts (if you're running the WSL version of the app, anyhow) to show kanji:  
sudo apt install -y fonts-noto-cjk fonts-noto-cjk-extra
